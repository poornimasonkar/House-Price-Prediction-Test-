# 🏠 Boston House Price Prediction

This project predicts house prices in Boston using Machine Learning and provides an interactive UI using Streamlit.

## 📌 Features
- Predict price based on 13 housing features
- Streamlit UI for input
- Feature importance graph
- Categorizes house as Affordable, Mid-Range, or Premium

## 💡 Tech Stack
- Python
- Pandas, NumPy
- scikit-learn
- Matplotlib, Seaborn
- Streamlit
